# Antonio’s House Decor Service

A simple interactive Python program that suggests interior design ideas based on user selections. Choose a room, select a preferred style, and receive a randomized furniture layout concept.

## Features
- Five room options: Living Room, Bedroom, Kitchen, Bathroom, Backyard
- Style selection for each room (e.g., Modern, Rustic, Scandinavian, etc.)
- Randomized furniture centerpiece and complementary item suggestions
- Optional “Surprise Me” mode

## How to Run
```bash
python house_decor.py
